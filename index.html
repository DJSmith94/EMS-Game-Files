RF Tug of War

Choose your RF battle

Configure your data carrier and data receive modem during prep, then use the spectrum analyzer to find the opposing transponder and build interference signals. First file to complete wins before the live timer expires.

PrepFive minutes to pick your data modem and set your own transponder plan.
BattleThe analyzer can expand to the full battle span. The chosen data modem stays selected, but the data RX must be retuned after carrier or format changes.
ScoringNormal: 2 Gb pseudo file with a 10 minute live cap. Eb/No, FEC loss, and receiver tuning control progress.

Using the selected mode preset.

Run the local server, then host or join from two computers on the same network.
